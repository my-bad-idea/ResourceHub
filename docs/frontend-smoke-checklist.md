# Frontend Smoke Checklist

## 初始化与认证

1. 首次进入未初始化环境，确认自动跳转 `#/setup`
2. 完成初始化后确认跳转首页并展示演示数据
3. 使用管理员账号登录，确认 Header 用户菜单和后台入口可用
4. 打开“忘记密码”，确认未知邮箱与已注册邮箱都返回统一提示
5. 使用邮件预览里的链接完成重置密码，确认无效 / 已使用 / 已过期 token 都能看到明确报错
6. 打开“修改密码”，确认当前密码错误、新旧密码相同、确认密码不一致时都有反馈

## 首页资源流

1. 搜索资源名称、描述、标签，确认列表实时过滤
2. 切换卡片 / 列表 / 时间轴视图，确认排序与展示正确
3. 通过类别、标签、快速访问筛选资源，再点击“清空筛选”恢复
4. 新增资源，确认可选类别、标签、可见性、启用状态全部生效
5. 编辑资源后刷新首页，确认更新内容与类别计数同步
6. 删除资源，确认列表、收藏、历史记录同步移除
7. 收藏 / 取消收藏资源，确认“我的收藏”立即更新
8. 连续访问资源，确认“最近访问”存在且最多保留 200 条

## 后台管理

1. 类别管理：新增、编辑、删除类别，确认资源计数与颜色展示正确
2. 标签管理：单个删除、批量删除，确认资源标签同步移除
3. 用户管理：新增用户、编辑用户、重置密码、删除用户
4. 删除拥有资源的用户后，确认资源仍保留且归属转移到执行删除的管理员
5. 系统配置：保存站点标题、Token 时长、注册开关、邮箱域名白名单
6. 邮件服务：保存 SMTP 配置；在 Mock 模式下确认“测试连接”弹出预览

## 响应式检查

1. 在 1280px 左右确认桌面布局正常
2. 在 768px 左右确认 Header、筛选栏、后台导航会换行或堆叠
3. 在 768px 以下确认首页卡片会自动缩列，后台表格可以横向滚动并保持可操作
